DocxFilesMerger CLI
Nouvelle version en ligne de commande

DocxFilesMerger est maintenant disponible en version ligne de commande (CLI) !
Cette interface simplifiée vous permet de lancer une démonstration ou d'obtenir des informations sur l'utilisation du script en mode terminal.

Lancer la démonstration
Utilisation en ligne de commande
Syntaxe de base
python docx_merger_cli.py [options]
Options disponibles
Option Description
-f, --fichier FICHIER Traiter un seul fichier ZIP
-d, --dossier DOSSIER Traiter tous les fichiers ZIP d'un dossier
-o, --output DOSSIER Dossier de sortie pour les résultats
-q, --quiet Mode silencieux (sans affichage de progression)
-r, --rapport FICHIER Générer un rapport CSV des résultats
-h, --help Afficher l'aide complète
Exemples d'utilisation

# Traiter un seul fichier ZIP

python docx_merger_cli.py -f archive.zip -o ./resultats

# Traiter tous les fichiers ZIP d'un dossier

python docx_merger_cli.py -d ./dossier_archives -o ./resultats -r rapport.csv

# Mode silencieux (sans affichage de progression)

python docx_merger_cli.py -f archive.zip -o ./resultats -q
Astuce : Pour obtenir de l'aide à tout moment, exécutez :
python docx_merger_cli.py --help
Fonctionnalités

Le système traite une archive ZIP contenant des documents (.doc ou .docx) et effectue les opérations suivantes :

  • Extraction de tous les fichiers .doc et .docx
  • Conversion automatique des .doc en .docx si nécessaire
  • Fusion de tous les documents en un seul fichier .docx
  • Génération d'une version PDF du document fusionné
  • Organisation des documents par ordre alphabétique
Important : Pour les grandes archives contenant de nombreux fichiers (5 000+), le traitement peut prendre plusieurs minutes.
Résultats du traitement

Après le traitement, vous obtiendrez :

  • Un document DOCX fusionné contenant tous les fichiers
  • Une version PDF du document fusionné
  • Optionnellement, un rapport CSV détaillant le traitement
